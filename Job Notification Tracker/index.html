About This System

This design system defines every visual and interaction rule for the KodNest Build System. It is the single source of truth. No decisions are made outside of it.

Design Rule
Never use a color, spacing value, or font that is not defined in tokens.css. If a token does not exist for your need, add it to tokens.css first.
